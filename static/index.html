Python Web Server Experiment

Implementasi pembelajaran socket programming dan HTTP server menggunakan Python

Proyek Pembelajaran Mandiri

Tentang Proyek Ini

Ini adalah eksperimen pembangunan web server dari dasar menggunakan Python dan modul socket. Proyek ini dibuat untuk memahami fundamental jaringan komputer, protokol HTTP, dan cara kerja web server secara mendalam.

Dikembangkan dengan pendekatan modular untuk mempelajari best practices dalam arsitektur perangkat lunak, dengan komponen terpisah untuk request handling, response building, dan connection management.

Python 3 Socket Programming Multithreading HTTP/1.1 Modular Design

Fitur Utama Server

  • Multithreading Architecture

    Mampu menangani beberapa koneksi klien secara simultan dengan implementasi threading yang efisien. Setiap koneksi ditangani oleh thread terpisah untuk memastikan responsivitas server.

  • Advanced HTTP Request Parsing

    Mem-parsing request HTTP dengan robust menggunakan RequestParser yang terpisah, mendukung metode GET dan headers. Dilengkapi dengan penanganan error yang baik untuk request yang tidak valid.

  • Dynamic Response Handling

    Membangun response HTTP yang valid dengan ResponseBuilder, termasuk status code 200, 404, dan 500. Mendukung Content-Type yang sesuai dengan file yang diminta.

  • Static File Serving

    Menyajikan file statis (HTML, CSS, gambar) dari direktori khusus dengan penanganan MIME types dasar. Dilengkapi dengan cache control untuk performa yang lebih baik.

  • Modular Design

    Struktur kode terorganisir dengan baik memisahkan concerns ke dalam modul terpisah untuk kemudahan pengembangan. Setiap komponen memiliki tanggung jawab yang jelas dan terisolasi.
